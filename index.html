Music Outputs with Machine Learning Algorithms

In these and similar projects, I try to compose original music using ML algorithms and artificial intelligence. For this reason, I research music theory and related areas and also learn the technical mathematics of art. Although this has been a field I have been interested in for a long time, I plan to build another project with these outputs, so my outputs are not public.

Below I tried to explain one of my music output processes:

On the technical side, I used a deep learning model called LSTM (I have also worked with Transformers, RNN and GANs). In the first step, I fed the model many MIDI files of classical piano pieces for training. By analyzing the note and chord sequences in these pieces, the model learned the basic rules and patterns of music, that is, which note tends to follow which. For technical knowledge about music I had to read a lot continuously. After training was completed, I gave the model a small seed melody and had it predict the notes one by one using the patterns it had learned, thus creating a completely original composition that had never been heard before. With ML algorithms and sometimes without using deep learning or machine learning techniques, using only Python code (MIDI libraries, music21, pyo, etc.) I produce various music outputs and edit them with applications such as Reaper.
